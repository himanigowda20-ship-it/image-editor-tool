# Image Editor Tool

An interactive image editing application built using Python, Pillow, and ipywidgets in Google Colab.

## Features
- Upload an image
- Adjust brightness, contrast, exposure, and saturation
- Real-time preview of changes
- Download the edited image

## Technologies Used
- Python
- Pillow (PIL)
- Matplotlib
- ipywidgets
- Google Colab

## How to Use
1. Run the notebook
2. Upload an image
3. Use sliders to adjust image properties
4. Click "Save & Download" to export the edited image
